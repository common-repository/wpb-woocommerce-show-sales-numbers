Show WooCommerce Show Sales Numbers.

== Description ==

By this simple plugin you can show how many times any WooCommerce Product sold. Can be set it to different location of single product page. Location can be set form this plugin settings. It's possible to disable showing the sales numbers if no sale. Also can be disable showing sales numbers for specific products.


> [DEMO](http://demo2.wpbean.com/?product=flying-ninja)  |  [Support](http://wpbean.com/support/)

### Plugin Features

* Two different Style for showing the sales number.
* Option for changing the placement of sales number.
* Option for disable showing sales number if no sales.
* Option for disable showing sales number for specific products.
* Very light weight.
* Work with all standard WordPress theme.
* Easy to use.
* Clean code & easy to customize.

== Installation ==

* Install it as a regular WordPress plugin

* Active the plugin.

* This plugin's will show the sale number in your single product page.

* Go to this plugin's setting panel for customize.



== Frequently asked questions ==

= How to change the placement of sale number? =
Go to this plugin setting page, here you will see a dropdown for selecting the placement.

= How can I change the font-size & color of WPB Show Product Sales Number for WooCommerce? =
Go to this plugin setting page, under style settings you will see color picker and font size settings.
